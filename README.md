## ✨ Fitur

| Modul | Fitur |
|-------|-------|
| **Dashboard** | Statistik total barang, pelanggan, petugas, transaksi |
| **Cashflow** | Laporan omzet per bulan dengan grafik |
| **Manajemen Barang** | Tambah, edit, hapus data handphone |
| **Manajemen Pelanggan** | Tambah, edit, hapus data pelanggan |
| **Manajemen Petugas** | Tambah, edit, hapus data petugas |
| **Transaksi** | Multi item (keranjang belanja), edit transaksi, hapus transaksi |
| **Cetak Nota** | Format struk penjualan yang bisa di-print |
